BTC Tracker

A Bitcoin-native website that tracks global assets priced in Bitcoin (BTC) terms. Watch everything trend to zero in BTC through interactive charts, return tables, and comparisons.

Features

Real-time Data

Live Bitcoin price and market data updates

Asset Categories

  • Stocks
  • Commodities
  • Metals
  • Indices

Interactive Charts

Multiple timeframe analysis with Highcharts

Return Analysis

Comprehensive performance tracking

Dark Theme

Bitcoin-inspired design with dark mode

Tech Stack

Frontend

  • Next.js 14 with TypeScript
  • React 18
  • Tailwind CSS
  • Highcharts

Backend & APIs

  • TradingEconomics API
  • Next.js API Routes

Getting Started

Step 1: Clone Repository
git clone https://github.com/your-username/btc-tracker.git
Step 2: Install Dependencies
npm install
Step 3: Configure Environment

Create a .env.local file with your API keys:

TRADING_ECONOMICS_API_KEY=your_api_key_here
Step 4: Run Development Server
npm run dev

Open http://localhost:3000 in your browser

Project Structure

  • src/
    • app/ - Next.js app router pages
    • components/ - Reusable React components
    • lib/ - Utility functions and API clients
  • public/ - Static assets
