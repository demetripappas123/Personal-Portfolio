Setting Up a Project with GitHub

Introduction

This instruction set will take you step by step through setting up a coding project with GitHub. GitHub is a very useful tool we will dive into more detail on, which allows us to track changes to coding projects, upload our work from a local device, and collaborate with others.

For this project, we will set up a project called Rock Paper Scissors, which will have an index.html file and a javascript.js file. We won't actually code the project itself, but we will create a secure link between our local device and the Git server, create the necessary directory and files on our local device, link our directory to a corresponding Git repository, and push (update) the files we created to it. This will set us up for using all of the tools Git has to offer for our project (in future instruction sets).

Operating System Requirements

These instructions are intended for Linux-based operating systems. The operating system (OS for short) is the software that runs your computer; when you think of using Windows versus a Mac, those are two different common operating systems.

For Windows or Mac users, you can use Oracle VirtualBox to set up a Linux environment. We also assume that you have:

  • Visual Studio Code installed
  • Google Chrome installed

Setting Up GitHub

GitHub Basics

GitHub is a platform for uploading and sharing your work, publicly hosting it on a server, and it has built-in version control– a system that allows you to track changes to your work, trace where errors might have occurred, and make changes on your own branches before merging them with the main project.

Go ahead and create a new account at github.com/signup if you don't have one already.

Creating an SSH Key

SSH (Secure Shell) is a safe way to link files and connect data to remote sites over unsecured networks. SSH connections will encrypt any data you link and allow you to push your local file changes to your GitHub account.

Step 1: Access the Command Line Interface (CLI)

Press Ctrl + Alt + T to open your terminal.

username@Xubuntu:~$
Step 2: Generate SSH Keys

Run the following command to generate your SSH key pair:

ssh-keygen -t ed25519 -C "your_email@example.com"
Step 3: Start SSH Agent and Add Key

Run these commands to enable automatic authentication:

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
Step 4: Display and Copy Public Key

Display your public key with:

cat ~/.ssh/id_ed25519.pub
Step 5: Add Key to GitHub
  1. Click your profile icon → Settings
  2. Select "SSH and GPG Keys"
  3. Click "New SSH Key"
  4. Paste your public key and save
Step 6: Test Connection

Verify your setup with:

ssh -T git@github.com

Creating the Project Files and Linking them to Git

Creating Project Files

Step 1: Create Project Directory

Create a new directory for your project:

mkdir rockPaperScissors
cd rockPaperScissors
Step 2: Create Project Files

Create the necessary files for your project:

touch index.html javascript.js

Linking to GitHub Repository

Step 1: Create GitHub Repository

Create a new repository on GitHub with the same name as your local directory.

Step 2: Link Local Directory to GitHub

Run these commands to link your local directory to GitHub:

git init
git remote add origin git@github.com:your-username/rockPaperScissors.git
Step 3: Push Files to GitHub

Add, commit, and push your files:

git add .
git commit -m "Initial commit"
git push -u origin main
